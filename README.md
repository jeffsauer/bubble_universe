# bubble_universe
Graphic animation demo using V language

https://hackaday.com/2026/07/19/qt6-brings-basic-to-the-web-browser-or-your-computer/

It showed a demo called "Bubble Universe" that I thought looked interesting. I thought it would be a good test for Google Gemini AI to see if it could convert the basic code into a native V language implementation using the gg module.
